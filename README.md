# Overview
ACNIVERSE is a deep learning-based acne detection system designed to identify multiple acne types from facial images and provide treatment recommendations. The project aims to assist users in understanding their skin conditions through automated object detection powered by computer vision technologies.
The system utilizes the YOLOv8 object detection model to detect and classify various acne categories from facial images. By combining artificial intelligence and dermatological knowledge, ACNIVERSE provides an accessible solution for preliminary acne analysis.

# Features
- Acne detection from facial images
- Multi-class acne classification
- object detection using YOLOv8
- Bounding box visualization
- Treatment recommendation based on detected acne type

# Acne Classes
1. Acne Scars
2. Blackheads
3. Nodules
4. Papules
5. Enlarged Pores
6. Pustules
7. Whiteheads

# Research Contribution
This project was developed as part of an Artificial Intelligence and Computer Vision initiative focusing on acne detection using deep learning techniques. It demonstrates the application of object detection models in healthcare-related image analysis.
